# 🎵 Song Pronunciation Trainer — Инструкция для русских пользователей

Веб-приложение для отработки произношения текстов песен на **английском, французском, немецком, итальянском, испанском, русском и китайском** языках. Работает прямо в браузере — ничего скачивать и устанавливать не нужно.

> **Откройте `index.html` в Chrome, Edge или Safari (требуется HTTPS или localhost).**

---

## 📋 Быстрый старт

| Шаг | Действие |
|-----|----------|
| 1 | Выберите язык песни сверху |
| 2 | Вставьте текст песни в текстовое поле |
| 3 | (Опционально) Вставьте перевод на английский |
| 4 | Нажмите **Start Practice** |
| 5 | Нажимайте 🔊 **Play** → 🎤 **Record** → ⏭ **Next** для каждой строки |

---

## 🔤 Выбор языка

В верхней части страницы — пять кнопок:

| Кнопка | Язык | Распознавание речи | Озвучка |
|--------|------|-------------------|---------|
| **English** | Английский | en-US | Английский голос |
| **Français** | Французский | fr-FR | Французский голос |
| **Deutsch** | Немецкий | de-DE | Немецкий голос |
| **Italiano** | Итальянский | it-IT | Итальянский голос |
| **Español** | Испанский | es-ES | Испанский голос |
| **Русский** | Русский | ru-RU | Русский голос |
| **中文** | Китайский | zh-CN | Китайский голос |

Нажмите на нужный язык — кнопка станет залитой цветом.

> **Для русских песен:** выберите **Русский**. Приложение будет распознавать ваше произношение как русскую речь и давать оценку. Имейте в виду, что SpeechRecognition для русского языка может работать менее точно, чем для английского — это особенность браузера, а не приложения. Если вы тренируете произношение на французском/немецком/итальянском/испанском, выберите соответствующий язык — оценка будет основываться на том, насколько точно распознавание засчитало вашу речь как слова этого языка.

## ⌨️ Пошаговая работа

### 1. Вставка текста

- **Lyrics**: вставьте текст песни (каждая строка — отдельная строчка)
- **English Translation** (необязательно): вставьте построчный перевод на английский — он будет показываться под текущей строкой
- Нажмите **Start Practice**

### 2. Режим тренировки

Появляется интерфейс из трёх зон:

#### Зона A — Телепромптер (дисплей строки)

- **Крупный текст** по центру — текущая строка
- Сверху (тускло) — предыдущая строка
- Снизу (ещё тусклее) — следующая строка
- Внизу счётчик: «Line 3 of 24»
- Если вы добавили перевод на английский, он показывается под текущей строкой курсивом

#### Зона B — Кнопки управления

| Кнопка | Действие | Когда нажимать |
|--------|----------|----------------|
| 🔊 **Play** | Озвучивает текущую строку голосом выбранного языка | Сначала — послушать эталон |
| 🎤 **Record** | Включает микрофон, слушает ваше произношение | После Play — повторить строку |
| 🔤 **Play Word** | Озвучивает одно слово подсветкой | Для отработки по словам |
| ⏭ **Next Word** | Переход к следующему слову | В паре с Play Word |
| ⏭ **Next** | Переход к следующей строке | После получения оценки |
| ⏭ **Finish** | (на последней строке) Показать итоги сессии | По завершении всей песни |

> 🎯 **Горячая клавиша:** Пробел (Space) — начать/остановить запись. Работает когда фокус не на кнопке и не на текстовом поле.

#### Зона C — Оценка и обратная связь

После записи появляется панель:

| Показатель | Что означает |
|------------|-------------|
| **Pronunciation Score** | Итоговая оценка в % (0–100) |
| **You said** | Что распознал микрофон (текст) |
| **Reference** | Оригинальная строка песни |
| **Confidence** | Уверенность распознавания % |
| **Accuracy** | Точность совпадения слов % |

Цвет оценки:
- 🟢 **≥ 80%** — отлично
- 🟡 **50–79%** — неплохо, есть над чем работать
- 🔴 **< 50%** — стоит повторить

Подсказки при низком балле:
- *«Check vowel sounds and word endings»* — проверьте гласные и окончания
- *«Slow down and focus on each syllable»* — замедлитесь и сфокусируйтесь на каждом слоге
- *«Try again and listen to the reference first»* — послушайте эталон ещё раз
- *«No speech detected»* — микрофон не зафиксировал речь, говорите громче

## 🎚️ Дополнительные настройки

### Пословный режим (Word by Word)

Кнопка 🔤 **Word by Word** включает режим, в котором кнопка **Play** озвучивает строку слово за словом с подсветкой каждого. Полезно для разбора сложных строк.

### Скорость озвучки (Speed)

Пять скоростей: **0.5x, 0.75x, 1.0x, 1.25x, 1.5x**

- 0.5x — медленно (для разбора)
- 1.0x — нормально (по умолчанию)
- 1.5x — быстро (как в настоящей песне)

## 🛠 Частые проблемы

| Проблема | Решение |
|----------|---------|
| Кнопка Record неактивна | Браузер не поддерживает SpeechRecognition. Используйте Chrome или Edge. |
| Появилось предупреждение жёлтым | SpeechRecognition не поддерживается в вашем браузере |
| «Microphone access denied» | Разрешите доступ к микрофону в настройках браузера |
| «No speech detected» | Проверьте микрофон, говорите громче и ближе |
| Запись прерывается | Не переключайтесь на другую вкладку во время записи |
| Русский язык распознаётся неточно | Это особенность браузера — SpeechRecognition для ru-RU менее точен, чем для en-US |

## 🎯 Советы для эффективной тренировки

1. **Слушайте оригинал** — всегда сначала нажимайте Play
2. **Повторяйте вслух** — записывайте своё произношение
3. **Следите за прогрессом** — шкала прогресса показывает, сколько осталось
4. **Используйте пословный режим** для трудных мест
5. **Замедляйте скорость** на сложных строках (0.5x–0.75x)
6. **Смотрите на перевод** — понимание смысла помогает произношению

Удачи в тренировке! 🎤🎶
